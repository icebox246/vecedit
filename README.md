# VecEdit

Editor for vector graphics.

## Quick start

Install:

- `raylib` >= 5.5
- `pkg-config`
- `cmake` >= 3.30
- `clang` or `gcc`

To setup build environment (manually):

```
mkdir build
cd build
cmake ..
```

To build and run (assuming current directory is already `build/`)

```
make -j8   # number relies on CPU count
./VecEdit
```
